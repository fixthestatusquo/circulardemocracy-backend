Bearer
query Parameters

string^\d+$

Messages

/api/v1/messages

Receives a citizen message, classifies it by campaign, and stores it for politician response

Authorizations:
Bearer
Request Body schema: application/json
external_id
required
string [ 1 .. 255 ] characters

Unique identifier from source system

sender_name
required
string [ 1 .. 255 ] characters

Full name of the message sender

sender_email
required
string <email> <= 255 characters

Email address of the sender

recipient_email
required
string <email> <= 255 characters

Email address of the target politician

subject
required
string <= 500 characters

Message subject line

message
required
string [ 10 .. 10000 ] characters

Message body content

html_content
string <= 50000 characters

HTML version of message content

text_content
string <= 50000 characters

Plain text version of message content

timestamp
required
string <date-time>

When the message was originally sent (ISO 8601)

channel_source
string <= 100 characters

Source system identifier

campaign_hint
string <= 255 characters

Optional campaign name hint from sender
